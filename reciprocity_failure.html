Reciprocity Failure

This project is adapted from a smoke effect shader I learnt online. I added some interactivity and color effects to make it my own. I used Meta Spark for this project, and Blender to create the icon and promo visuals.

Try On Instagram

Concept

Concept

This effect aims to encourage the user to engage their bodies. The majority of Instagram effects apply effects on the face alone, so I wanted to make an interactive smoke effect.

This effect aims to encourage the user to engage their bodies. The majority of Instagram effects apply effects on the face alone, so I wanted to make an interactive smoke effect.

How It Works

How It Works

Meta Spark provides a person segmentation mask isolating people from the background. This is fed into the smoke shader, which works by employing a feedback loop where the previous frame is translated and distorted and and fed into the next iteration to produce a continuous effect. I learnt the smoke shader from this tutorial. Finally, I used shaders to invert the colors and burn the image for a unique look.

Interactivity

Interactivity

I wanted to allow the user to control the smoke in an intuitive way. I added swipe interactions, where the smoke travels in the opposite direction of the swipe. Tapping the screen turns off the smoke and “pulses” the screen to communicate the change to the user.
