FaceMeshGL

Developed a face filter app for augmenting makeup, tattoo, and face mask effects similar to Instagram and Snapchat's. This was my semester project for COMP477 Animation For Computer Games. It's built using C++, OpenGL, OpenCV, and Dlib.

Github Page

Motivation

Motivation

I used my final courses in university as an excuse to keep exploring augmented reality. I decided to create my own AR face effect app to better understand how social media app's AR tech works on the backend. Also, making my own app could theoritically free me from the restrictions of social media apps.

Face Detection

Face Detection

I used a C++ library called Dlib which contains a pre-trained face detection model. It can predict 68 landmarks on the human face, which outline key areas like the eyes, mouth, nose, and jawline in 2D space. I use these 68 points to construct a mesh that follows the face.

Putting It Together

Putting It Togethers

I used OpenCV to capture frame data from the webcam and use it as a texture in OpenGL. The background webcam feed is a plane rendered in orthographic projection at the very back of the viewing volume, scaled to fit the window size.

Challenges

Challenges

One issue was working between OpenCV and OpenGL’s coordinate systems, which was a matter of handling the change in resolution from the webcam to the OpenGL window and flipping the Y-axis. I also tried to improve performance by scaling the camera frame down when detecting the face landmarks, and skipping frames since faces don't move fast enough to require updating every frame.

Final Touches

Final Touches

Predicting a slightly different point based only on the current frame produces shaky results. I stabilized it by using Optical Flow techniques where we factor in motion between the previous and current frames to get smoother motion. Finally I painted several textures to have fun with my own "filter" app.
